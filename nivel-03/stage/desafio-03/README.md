## 💻 O desafio

Space Cream é uma página web de uma loja fictícia de sorvetes. Nesse projeto aprendi um pouco sobre:
* Novas tags semânticas;
* Display grid;
* Responsividade;
* Uso do *clamp* para fontes responsivas;
* *Media queries*;
* Variáveis CSS;
* Animações e transições CSS;
* Mobile First.
