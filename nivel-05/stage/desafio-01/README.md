## 💻 O desafio

O Jogo da adivinhação é uma página web com um programa que sorteia um número de 0 a 10 e mostra uma mensagem caso o usuário acerte o número. Nesse projeto aprendi um pouco sobre:
* Manipulação da DOM;
* Eventos e callbacks;
* Biblioteca JS Math();
* Básico de refatoração.
