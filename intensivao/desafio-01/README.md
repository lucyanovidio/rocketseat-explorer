## 💻 O desafio

Neste desafio a ideia foi rever conceitos de HTML e CSS que foram/seriam vistos nos Stages dos níveis do 1 ao 4. Aplicamos:
* Flexbox
* CSS Grid
* Variáveis CSS
* Mobile First
* Básico de responsividade
* Acessibilidade e tags semânticas
