## 💻 O desafio

Treine.me é uma página web simples para uma empresa fictícia de treinos e instrução de exercícios físicos. Nesse projeto aprendi um pouco sobre:
* Semântica HTML;
* Diferentes propriedades de *position* no CSS;
* *Inline* vs *Block*.
* Flexbox;
* *Linear-gradient*.
